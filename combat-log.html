Combat Log

Upload an Ikusa Logger export to see kill/death detail from one player's viewpoint.
Ikusa is a third-party combat logger some members run alongside BDO — it watches the game's network traffic and writes a plain-text .log file of every kill/death that player's client saw ("PlayerA has killed PlayerB from GuildName"). Drop that file below to parse it — everything happens locally in your browser; nothing is uploaded anywhere.
⚠ This only reflects one client's view, not a full war roster — it's not a substitute for /addwar or the official War Scores record, just a detail lens on individual engagements.
